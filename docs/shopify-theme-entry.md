# 在 Shopify 主题中加入「珠宝定制」入口

定制应用部署后的公网地址示例：`https://diy.lilpeb.com/builder`（请替换为你的 Vercel 域名或自定义子域）。

## 方式 A：导航菜单外链（推荐，最快）

1. Shopify 后台 → **在线商店** → **导航**。
2. 编辑 **主菜单**（或页脚菜单）。
3. **添加菜单项**：
   - 名称：例如「手串定制」「DIY」。
   - 链接：选择 **外部链接**，填入 `https://diy.lilpeb.com/builder`（或你的生产 URL + `/builder`）。
4. 保存。

顾客从店铺任意页点击该菜单即可进入 Next.js 定制页；结账仍通过应用内创建的购物车跳转 Shopify Checkout。

## 方式 B：首页按钮（主题编辑器）

1. **在线商店** → **主题** → **自定义**。
2. 在首页添加 **按钮** 或 **图片横幅** 的链接，指向上述完整 URL。
3. 保存。

## 方式 C：App Proxy（同主域路径）

若希望地址形如 `https://lilpeb.com/apps/jewelry-diy/...`，需在合作伙伴后台创建应用并配置 **App proxy**，由 Shopify 将路径代理到你的服务器；实现与验签比 A/B 复杂，可在 MVP 验证后再做。

## 移动端

方式 A/B 对移动端主题同样生效；无需单独配置，只要链接为 `https` 即可。
